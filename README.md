# Bot_BLUM

Enjoy with automation BOT

### How to use

1. Clone Repository
`git clone https://github.com/syauqqii/Bot_BLUM`

2. Open Folder
`cd Bot_BLUM`

3. Install Dependencies
`npm i`

4. Run application
`npm start`
